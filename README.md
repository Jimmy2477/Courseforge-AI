# CourseForge AI 🎓

An AI-powered courseware generation platform for Learning Management Systems (LMS).

## 🌐 Live Demo
**[https://courseforge-ng.netlify.app](https://courseforge-ng.netlify.app)**

## 📌 What It Does
CourseForge AI allows educators to generate complete, structured course modules in minutes by simply entering a topic. The platform uses AI to create full lessons with objectives, content, activities and assessments.

## ⚡ Key Features
- AI-powered course generation using Groq LLM
- Web-enriched content via Tavily real-time search
- Module review system — approve, reject or regenerate
- Google Drive auto-save for approved modules
- User authentication with free (5/day) and Pro tier
- Fully deployed on cloud — works 24/7 without local machine

## 🛠 Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, JavaScript |
| Automation | n8n (Railway cloud) |
| AI Model | Groq LLM (llama-3.1-8b) |
| Research | Tavily Search API |
| Database | Supabase (PostgreSQL) |
| Storage | Google Drive API |
| Hosting | Netlify + Railway |
